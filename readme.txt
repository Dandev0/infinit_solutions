Тестирование API запроса для создания компании.

В conftest.py производим подготовку:
1)Делаем запрос в функции start_request
2)Преобразовываем JSON в функции reformat

В test_main.py прописаны все необходимые тесты:
1) Статус код
2) Успех в выполнении запроса
3) Проверка равенства переданного нами имени и созданого
4) Проверка равенства переданных типа компании и пользователей с полученными. 


Данное тестирование проходило на реальном API запросе, но я его скрыл, чтобы не баловались)))
